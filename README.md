# 1. 阅读背景
最近也在做更多软件构建相关的工作，很多东西没有一个主要的系统性流程，存在很多内容的缺失，知识点在实战中，也存在质量性的问题，所以决定阅读一下这本书，并记录下学习的笔记。


在作者的这本书中，前言部分，也告知了读者群体及对应群体的着重点、收益等。
- 读者群体
  - 经验丰富的程序员
  - 技术带头人
  - 自学者
  - 学生

- 可以得到的收益
  - 写出更好的程序
  - 全面的软件构建参考
  - 随时备用的核对表
  - 以更广的视角检视软件开发
